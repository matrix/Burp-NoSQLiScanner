This extension provides a way to discover NoSQL injection vulnerabilities.

The main features are:

  • Add Passive and Active Scanner checks
  • Try to identify NoSQLi
  • Try to identity SSJI using experimental payloads
  • Manipulate the standard Insertion Points identified by Burp, in such a way as to reduce user-induced errors.

This extension requires Java version 8
